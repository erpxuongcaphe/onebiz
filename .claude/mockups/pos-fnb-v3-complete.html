📋 POS F&B v3 — Phương án theo feedback CEO

Em đã tổng hợp lại 5 yêu cầu của anh: (1) Sidenav ẩn ☰ desktop, (2) Sơ đồ bàn vị trí riêng, (3) Danh mục cột trái HOẶC ngang wrap, (4) Mobile/Tablet không kéo ngang, (5) Customization toàn diện.

1️⃣ Desktop — Sidenav ẨN ☰ + Danh mục cột trái

Feedback CEO: "Menu trái nên được ẩn trong biểu tượng 3 gạch khi nào muốn thì mở lên, trừ sơ đồ bàn thiết kế chỗ để cho phù hợp. Danh mục nên để qua cột trái cho dễ xem hơn."

📌 Cách giải quyết

  • Sidenav (Bán hàng / Bếp / Đơn / Báo cáo / ...) → ẩn dưới icon ☰ ở góc trái header. Click mở slide drawer 280px overlay (không push content).
  • Sơ đồ bàn / Bán hàng → tách view-toggle riêng ngay header (giống KiotViet "Phòng bàn / Thực đơn"). Bấm switch instant, không qua sidenav.
  • Danh mục (Cà phê / Trà / Sữa...) → cột trái 200px. Mỗi category 1 row có icon + tên + count. Sub-category nested expand/collapse khi click parent.
  • Menu grid 5 cols (đủ rộng vì sidenav đã ẩn) → cards lớn dễ bấm.
  • Cart 400px bên phải giữ nguyên.

6️⃣ Customization toàn diện — Em check + đề xuất

Feedback CEO: "Có 1 điều em check xem đã có thể tùy biến màn hình pos chưa? Ví dụ: sơ đồ bàn, khu vực, số lượng bàn, sắp xếp thực đơn theo ý, bố cục thực đơn theo ý."

📊 Trạng thái hiện tại — Backend đã sẵn 80%

✅ Vị trí bàn (X, Y)

DB schema có position_x, position_y. Cần build UI drag-drop editor.

✅ Khu vực (zones)

DB schema có zone + sort_order. UI list zones có nhưng chưa drag-drop layout.

✅ Sort categories

DB có categories.sort_order + RPC swap. UI có button up/down cơ bản, cần upgrade drag-drop.

✅ Sort products

DB có products.sort_order. CHƯA có UI drag-drop reorder menu.

❌ Layout config (bố cục)

Chưa có. Cần thêm tenant_settings.pos_layout JSON: cart left/right swap, density, theme color.

❌ Background image

Sơ đồ bàn chưa support upload ảnh nền (vẽ layout quán) làm guide cho drag-drop.

🎯 Em đề xuất 4 trang Customization mới

  1. 📍 Sơ đồ bàn editor (mockup ở mục 3 phía trên) — drag-drop bàn, resize, multi-zone, snap lưới, ảnh nền tham khảo.
  2. 🍽 Menu reorder editor — drag-drop categories + products, set "best-seller" sticky top, hide/show theo ca/giờ.
  3. 🎨 Theme & Layout config — primary color picker, cart left/right, compact/comfortable density, hide/show panels.
  4. 🪑 Zone manager — thêm/xoá khu vực, đổi tên (Sảnh 1 → "Tầng trệt"), bulk-create bàn (vd "Tạo 8 bàn cho Sảnh 2").
